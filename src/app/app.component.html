Description
Angular modal component
  • - LcModal component is an Angular component for displaying custom content in modal
  • - Show any Angular component in modal
  • - Supports multiple modals and modals inside of a modals
  • - Has maximize/restore
  • - Resizable and draggable
  • - Compatible with Angular 2+ up to Angular v8.0.0
  • - Only dependency is Font Awesome
Getting Started
  • 1. Fork and clone this repo
  • 2. npm install
  • 3. Open a terminal and type "npm run start"
  • 4. Open "http://localhost:4200" in browser
Installation
To install this component to an external project, follow the procedure:

  • 1. Install lc-modal package from npm:

    npm install @libusoftcicom/lc-modal

  • 2. Make a component

    
import { BaseModalComponent } from '@libusoftcicom/lc-modal';

@Component({
   template: `
   <form (ngSubmit)="confirm(model)"></form>
      <button type="submit">Submit</button>
   </form>
   `
})
export class ModalComponentExample extends BaseModalComponent {}

  • 3. Register ModalModule in NgModule with components that are opened in modal:

    
   import { ModalModule } from '@libusoftcicom/lc-modal';
   import { ModalComponentExample } from '...'; // component used in modal

    @NgModule(
      {
        declarations: [...],
        imports: [
          ModalModule.withComponents([ModalComponentExample]),
          ...
        ],
        providers: [...],
        bootstrap: [...]
        })
    export class AppModule {}

  • 4. Add modal anchor component

    <dialog-anchor></dialog-anchor>

  • 5. Open modal

    
    let modalResult = await this.modal
      .title('Example modal')
      .component(ModalComponentExample)
      .setHeight(370)
      .setWidth(700)
      .draggable(true)
      .showMaximize(true)
      .open();
Config options 

this.modal
  .title('Example modal')
  .setHeight(370)
  .setWidth(700)
  .additionalParams(  { } || [] || String || Number || Boolean)
  .draggable(false)
  .resizable(true)
  .closeOnClick()
  .offsetLeft( 200 )
  .offsetTop( 200 )
  .closeOnError()
  .preClose(() => Promise.resolve() )
  .showClose( true/false )
  .confirm( { data } )
  .cancel()
  .setClass('test-class')
  .removeClass('test-class')
  .component(ModalComponentExample)
  .draggable(true)
  .showMaximize(true)
  .open();
title
Set modal title.
setHeight
Set modal height.
setWidth
Set modal width.
draggable
Enable/disable changing modal position.
resizable
Enable/disable changing modal size.
closeOnClick
Enable/disable closing modal on background click.
offsetLeft
Set modal position from left side of browser window.
offsetTop
Set modal position from top of browser window.
closeOnError
Enable/disable closing modal on error.
preClose
Set function that is executed before closing modal.
showClose
Show/hide modal close button.
confirm
Close modal with confirmation.
cancel
Close modal without confirmation.
setClass
Set custom css class on modal.
removeClass
Remove custom css class on modal.
component
Set modal component.
draggable
Enable/disable modal dragging.
showMaximize
Show/hide modal maximize button.
open
Open modal.
